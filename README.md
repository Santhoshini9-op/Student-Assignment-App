# Assignment Management System

## Overview

Assignment Management System is a mobile UI/UX project designed in Figma for students and teachers to manage academic assignments in one place.

Students can view assignments, check deadlines, submit work, and view marks. Teachers can create assignments, track submissions, and manage marks through a separate dashboard.

---

## Problem Statement

Students may miss assignment deadlines or struggle to track pending work and marks. Teachers also need a simple way to create assignments and manage student submissions.

---

## Solution

This project provides separate student and teacher workflows through role-based dashboards. It helps organize assignment details, deadlines, submissions, and marks in a simple mobile interface.

---
